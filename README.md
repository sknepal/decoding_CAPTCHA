## Introduction

A simple approach to breaking a weak text based CAPTCHA using Image Processing and Neural Network. For more details, [read the blog post](http://www.thelacunablog.com/breaking-text-based-captcha.html).


## Requirements
* Python 2.7
* Ipython notebook
* Pillow
* Scipy
* Numpy
* Dill
* Pytesseract
* Selenium
* Pandas
